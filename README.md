# 📌 About Pandas

**Pandas** is one of the most powerful and popular Python libraries used for:

- 📊 Data Analysis
- 🧹 Data Cleaning
- 🔄 Data Transformation
- 📁 File Handling
- 📈 Exploratory Data Analysis (EDA)

It provides fast and flexible data structures:

- 🐼 **Series** → One-dimensional labeled data
- 🗂️ **DataFrame** → Two-dimensional table-like data
